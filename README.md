# Session Capacity Monitor (Python)

## Overview
This project simulates real-world session capacity monitoring for infrastructure environments.

## Features
- Calculates session utilization percentage
- Compares against defined threshold
- Triggers alert when exceeded
- Generates CSV report
- Visualizes usage vs threshold using matplotlib
